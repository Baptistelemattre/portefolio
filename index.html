Projet Marathon

Projet de développement en 48 heures d'une application web

Projet du semestre 3 à l'IUT de Lens : nous avions à réaliser un site internet permettant de référencer des séries télévisées. Il a été réalisé en PHP avec le Framework Laravel et inclut diverses fonctionnalités comme afficher les détails d'un épisode ou d'une série dans son entièreté. Ce projet a été mené en collaboration avec les DUT Multimédia et les DUT Informatique.

arma III

Jeu de simulation de guerre.

Ce projet a été réalisé en parfaite autonomie sans aucune aide (professeurs, cours…). Il a nécessité le développement d’Item 3D afin de pouvoir l’utiliser comme un support de jeu de rôle. J’ai ainsi développé une certaine rigueur permettant l’optimisation de chaque Item et accéder à une meilleure expérience de jeu. Dans le cadre de ce projet, j’ai aussi appris à utiliser les logiciels Photoshop, Object Builder (logiciel de la suite Arma 3, dédié à la création 3D), ainsi que le langage de programmation C appliqué au jeu Arma 3 avec du CPP (application de textures sur les objets).
